TumblrでのGoogleXML Sitemapの場所
設置場所 google Sitemap tumblr XML
Published: 2014-06-27
これは自動的に作られている様だがわかりにくいので備忘録。

筆者のサイトで以下のサイト、

IT.PC.MobileInfomation

このTumbleブログは独自ドメイン運営しています。

まぁそれは置いておいて、独自ドメイン/sitemap.xmlを打つと一応サイトマップらしきものが表示されます。 

 This XML file does not appear to have any style information associated with it. The         document tree is shown below.

    <sitemapindex xmlns=“http://www.sitemaps.org/schemas/sitemap/0.9">
    <sitemap>
    <loc>http://realtimeit.2nen.com/sitemap1.xml</loc>
    </sitemap>
    </sitemapindex>

 

でも、これってXML Sitemapの場所を示しているんですね＾＾
ここに書かれている様にサイトマップのアドレスは、sitemap1.xmlとなるのですね。
